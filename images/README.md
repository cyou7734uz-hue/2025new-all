# 圖片資料夾

此資料夾存放專案所使用的圖片檔案。

## 必需圖片

### Medusa 頭像
- **檔案名稱**：`medusa.png`
- **說明**：頁面四周隨機漂浮的 Medusa 角色圖片，點擊後會淡出消失 0.5 秒，隨後隨機出現在頁面四周其一。
- **建議尺寸**：180×180 像素或更大（會根據 `medusaSize` 縮放）
- **格式**：PNG（支援透明背景）

## 其他浮動圖片（可選）

如果要新增多個漂浮圖片，可在此資料夾內加入以下檔案，並在 `sketch.js` 中的 `astroImgNames` 陣列新增對應的檔案名稱：

- `penguin.png` - 企鵝
- `angel.png` - 天使
- `baby.png` - 嬰兒
- `unicorn.png` - 獨角獸
- `blue_snake.png` - 藍色蛇
- `yellow_snake.png` - 黃色蛇

## 使用方法

1. 將圖片檔案放在此資料夾內
2. 在 `sketch.js` 中確保檔案名稱與實際檔案相符
3. 若圖片無法載入，瀏覽器控制台會顯示警告訊息
